scrapwell

scrapwell is a minimal MCP memory server for LLM agents. No hidden LLM calls. No server to run. Just human-readable Markdown files on disk — fully searchable.

Get started

Open-source MIT License. MIT License

No hidden LLM calls

The calling LLM decides what to save and how to classify it. scrapwell just stores and indexes — zero extra API cost per save.

Markdown as source of truth

All memories are plain Markdown files on disk. Browse, edit, or grep them in any text editor. No proprietary database, no lock-in.

Obsidian compatible

Open the memory directory as an Obsidian vault. [[wikilinks]] resolve automatically and the graph view shows knowledge connections.

10 MCP tools over stdio

Full CRUD for entities and documents, full-text search with highlighted snippets, tree browsing, and index rebuild — all over MCP stdio.

Pluggable search backend

Swap between LanceDB (default) and Tantivy via a single config field. Both backends are abstracted behind a trait.

Entity → Topic → Document

A simple three-level hierarchy keeps knowledge organized without enforcing rigid schemas. Topics are optional and created on demand.
